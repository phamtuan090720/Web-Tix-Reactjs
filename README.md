# Đồ Án Cuối Khóa Cybersoft Booking Movie.
 Đồ án này được thực hiện bởi **Phạm Tuân** và **Trần Hải Bình**.

 Tham Khảo Layout tại https://tix.vn/.
 
  Phân Chia Công Việc Như Sau:

**Phạm Tuân:**

- Home
- Login
- Register
- Thông Tin Cá Nhân
- Detail movie
- Booking ticket

**Trần Hải Bình:**

- Dashboard
- Booking Manager Page
- User Manager Page
- MovieManagerPage
- TheaterManagerPage
